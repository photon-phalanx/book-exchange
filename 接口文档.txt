用户相关

注册 
url: /user/register
method: POST
客户端发送一个对象
uid：学号，唯一
pw：密码
name: 姓名
email:
phone:
如成功，后台返回空对象，否则返回失败原因

登录
url: /user/login
method: POST
客户端发送一个对象
uid
pw:密码
如成功，后台返回需要的信息（应该就是整个个人注册信息），否则返回失败原因

查询自己借的书
url: /user/borrow
method: POST
客户端发送
{
     rdate:  // 最后的那个书的_id
     length: 5  // 每页显示个数
}
如成功，后台返回同下

查询自己发布的书
url: /user/publish
method: POST
客户端发送一个对象
{
     id:  // 最后的那个书的_id
     length: 5  // 每页显示个数
}

服务器回传的参数需为: [], // 表格数据

如成功，后台返回对象数组，否则返回失败原因

{
名称 title
发布时间 stime
owner
reader
应还日期rdate
分类 type 1教科书 2小说 3科学书 4工具书 5其他书
类型 flag 1送 2借 4both
借出日期 ldate
图片位置 path
续借标志 renewflag  0未续借 1已续借
处理中状态以rdate为-1表示吧……
}
