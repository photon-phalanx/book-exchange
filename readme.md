
## 目录结构

```
.
├── README.md           
├── dist               // 项目build目录
├── index.html         // 项目入口文件
├── package.json       // 项目配置文件
├── src                // 生产目录
│   ├── assets         // 图片等静态资源
│   ├── components     // 各种组件
│   ├── methods        // 需要复用的函数
│   ├── views          // 各种实际的页面
│   ├── filters.js     // 各种过滤器
│   ├── routers.js     // 各种路由
│   └── main.js        // Webpack 预编译入口
└── webpack.config.js  // Webpack 配置文件
```
